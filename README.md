# Autocomplete TS ⚛️

### 💻 &nbsp; Steps to run the project 
- Clone the repository and run sequentially into it: <br />
```yarn install``` <br />
```yarn start``` <br />
- Enjoy it in localhost:3000 🚀 
<br />
- Type some letters like `pa`, `an` in input and see the results
